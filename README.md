The lab consists of 24 programs designed to build understanding of Python programming concepts. The first set of programs focuses on conditional statements, where you check whether a number is positive, negative, or zero, determine if it is even or odd, find the largest among two or three numbers, assign grades based on marks, and create a simple menu-driven calculator using if-elif conditions. The next group introduces loops, including programs to print numbers from 1 to 10, display even numbers within a range, calculate the sum of the first 10 natural numbers, generate multiplication tables, use a while loop to print numbers, compute the sum from 1 to n, and repeatedly take input until the user enters zero. The following section demonstrates the use of break and continue statements, such as stopping a loop when a number reaches 6, skipping a specific number, and searching for an element in a list. The fourth part focuses on functions, including defining functions to print “Hello, World!”, calculate the square of a number, add two numbers, check whether a number is even or odd, compute factorial, and display multiplication tables. Finally, the integrated programs combine multiple concepts, such as using functions and loops together to check even or odd numbers from 1 to n, and building a menu-driven calculator that continues running until the user chooses to exit.
